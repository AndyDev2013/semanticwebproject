Semantic Web Project - Galway Geolocation Information
  • API and Help Raw Database ID's File Data - Galway Attractions File Data - Galway Parks File Data - Scenic Routes File Data - Galway Protected Structures File Data - Galway Recreation Strategy
About
The purpose of using information from these five data sets was to show scenic routes, attactions and parks from around Galway. This kind of information could be used in the tourism sector to generate walking tours, bus tours and other excavations around Galway county.
GitHub Project Link for more detail
Instructions
All data is returned in raw geojson format as text. To query, click on any of the Url links to copy the data or make a request to the given urls. Below is a key showing all of the available functions with the api.
Request Type Description Example Url
GET Get the original Galway Attractions Data from FILE http://127.0.0.1:8000/GET_originaldata_galway_attractions
GET Get the original Galway parks Data from FILE http://127.0.0.1:8000/GET_originaldata_galway_parks
GET Get the original Galway scenic Data from FILE http://127.0.0.1:8000/GET_originaldata_galway_scenic
GET Get the original Galway structures Data from FILE http://127.0.0.1:8000/GET_originaldata_galway_structures
GET Get the original Galway recreation strategy Data from FILE http://127.0.0.1:8000/GET_originaldata_galway_recreationstrat
GET All geojson data entries from database http://127.0.0.1:8000/GET_allData
DELETE
[Warning]
Clears the entire database. You will have to restart the server again to re-populate the pouchdb		 http://127.0.0.1:8000/DELETE_allData
