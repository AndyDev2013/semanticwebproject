Semantic Web Project - Galway Geolocation Information
  • API and Help File Data - Galway Attractions File Data - Galway Parks File Data - Scenic Routes
About
The purpose of using information from these three data sets was to show scenic routes, attactions and parks from around galway. This kind of information could be used in the tourism sector to generate walking tours, bus tours and other excavations around Galway county.
